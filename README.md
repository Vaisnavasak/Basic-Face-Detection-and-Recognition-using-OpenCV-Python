# Basic-Face-Detection-and-Recognition-using-OpenCV-Python
This is a model that detects the faces and recognize it based on the database generated. It is basically a 3 step process, which incudes
                                 1. Generating dataset
                                 2. Training based on the dataset
                                 3. Detect the face and name it if it is already in the dataset
In addition to this, it also greet the person once it recognize the person

**OBJECTIVES**
 1. It generates it's own dataset and train from it
 2. In an improved version it can be implemented in any kinds of simple and basic face recognition system
 3. It also greet and read aloud the name of the person
 4. It can be a better alternative in case of any fault in face recognizer machines used in offices etc...
